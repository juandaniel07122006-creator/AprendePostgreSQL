¿Qué es PostgreSQL?

Historia y Naturaleza

PostgreSQL es un potente Sistema Gestor de Bases de Datos Objeto-Relacional (ORDBMS) de código abierto (Open Source). Posee más de 35 años de desarrollo activo impulsado por una sólida comunidad global.

Se destaca por su arquitectura probada, fiabilidad asombrosa, integridad de datos y correctitud, manejando desde pequeñas aplicaciones web hasta enormes clústeres de datos (Terabytes y Petabytes).

Características Clave e Importancia

  • ✓ Cumplimiento ACID: Garantiza Atomicidad, Consistencia, Aislamiento y Durabilidad en las transacciones para nunca perder datos.
  • ✓ Concurrencia (MVCC): Utiliza el Control de Concurrencia Multiversión, lo que significa que las lecturas no bloquean las escrituras y viceversa, manteniendo un alto rendimiento.
  • ✓ Tipos de datos avanzados: Soporta no solo esquemas relacionales, sino formatos documentales nativos como JSON y JSONB.
  • ✓ Fiabilidad (WAL): Usa el Write-Ahead Logging (WAL) para recuperación de desastres (Point-in-time recovery).
